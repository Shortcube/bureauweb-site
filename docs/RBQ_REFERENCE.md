# RBQ_REFERENCE - BureauWeb (v1.0)
Derniere mise a jour: 2026-01-19

Objectif: liste interne de metiers/services pour lesquels le RBQ est requis, maintenue en parallele de l'onglet Liste (colonne B).

Liste RBQ requis:
- Asphaltage
- Balcons / terrasses
- Beton
- Carrelage
- Charpente
- Climatisation
- Clotures
- Clotures et portails
- Construction generale
- CVAC / Chauffage
- Demolition
- Ebenisterie
- Electricite
- Electricite industrielle
- Excavation
- Ferronnerie
- Fondations
- Gaz naturel
- Gicleurs / incendie
- Gypse / tirage de joints
- Isolation
- Maconnerie
- Menuiserie
- Pavage
- Peinture
- Piscines
- Planchers
- Plomberie
- Portes et fenetres
- Refrigeration commerciale
- Renovation commerciale
- Renovation residentielle
- Revetement exterieur
- Soudure
- Spas
- Toiture
- Ventilation

Regle:
- Si le secteur n'est pas dans cette liste: RBQ requis = Non par defaut.
- Exception: si l'activite ressemble a "Construction generale" sous un libelle non standard, exiger validation manuelle (unknown) plutot que supposer.
